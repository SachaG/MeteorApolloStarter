# Meteor + Apollo + React starter kit


## Setup & run :
1. Clone repo
2. cd MeteorApolloStarter
3. meteor npm install
4. meteor


## Features :
* Apollo 2.0
* React Router 4
* Accounts: password and google, login & registration in GraphQL
* SSR
